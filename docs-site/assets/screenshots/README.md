# Screenshot Pipeline

Store documentation screenshots in this folder.

Recommended naming:

- `guided-mode-dialogue-overview.png`
- `guided-mode-quest-overview.png`
- `validation-panel-blocking-issue.png`
- `simulation-panel-choice-node.png`
- `default-dialogue-widget-runtime.png`
- `quest-log-runtime.png`

Recommendations:

- capture both editor and runtime examples
- crop tightly around the subject
- prefer 16:9 editor captures
- annotate screenshots in the docs page rather than baking arrows into every image when possible
